FinMK — AI-Powered Financial Intelligence Ecosystem

Comprehensive finance platform using Django + React + MongoDB with a nested schema for privacy-first data handling and real-time dashboard. Integrates Amazon Chronos for 30-day spending forecasting, bart-large-mnli for automated transaction categorization, a Real-Time Voice Assistant, and a Google Gemini RAG Chatbot.
